# Hash

## 해시 테이블

자료구조 종류 중 하나로 key 와 value를 가짐.(배열과 차이점이라고 한다면 key 값이 배열은 숫자만 가능하다는 점) 해시 테이블은 숫자 뿐 아니라, String도 key 값이 가능하다.
이 방법은 원소의 검색뿐 아니라 추가, 삭제도 효율적으로 수행할 수 있다.
